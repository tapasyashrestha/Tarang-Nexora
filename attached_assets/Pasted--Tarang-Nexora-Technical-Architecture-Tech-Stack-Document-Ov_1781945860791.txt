# Tarang-Nexora – Technical Architecture & Tech Stack Document

## Overview

Tarang-Nexora is an advanced Strategic Foresight and Consequence Modeling Platform that helps e-commerce businesses understand the ripple effects of decisions such as price changes, inventory adjustments, and marketing spend modifications.

The system combines machine learning, simulation modeling, dependency mapping, and interactive visual analytics.

# API Endpoints

## Dashboard

GET /api/dashboard

Returns:

* Revenue
* Orders
* Ratings
* Delivery Metrics

---

## Forecast

POST /api/forecast

Input:

{
"price_change": -15
}

Output:

{
"expected_sales": 12500,
"expected_revenue": 95000
}

---

## Scenario Simulation

POST /api/simulate

Input:

{
"price_change": -20,
"marketing_change": 10
}

Output:

{
"sales_growth": 18,
"inventory_risk": 72,
"rating_change": -0.3
}
